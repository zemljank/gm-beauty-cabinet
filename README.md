# GM Beauty Cabinet

Каталог уходовой косметики и космецевтики с акцентом на премиальную подачу, фильтрацию и удобное пополнение из прайса.

- Live demo: [gm-beauty-cabinet.vercel.app](https://gm-beauty-cabinet.vercel.app)
- Репозиторий: [github.com/zemljank/gm-beauty-cabinet](https://github.com/zemljank/gm-beauty-cabinet)

## Что уже есть

- главная страница с hero, категориями и блоком `Выбор врача`;
- отдельный каталог с поиском, фильтрами и сортировкой;
- карточка товара, корзина и раздел `Избранное`;
- локальные фотографии товаров и баннеров;
- структура данных, которую можно расширять без правки интерфейса.

## Запуск

```bash
npm install
npm run dev
```

## Проверка

```bash
npm run build
npm run lint
```

## Где добавлять товары

Все позиции лежат в файле:

`src/data/products.ts`

Чтобы добавить новый товар, нужно добавить новый объект в массив `products`:

```ts
{
  id: "unique-id",
  brand: "ANGIOPHARM",
  country: "Россия",
  name: "Название продукта",
  price: 2500,
  volume: "50 ml",
  category: "cream",
  concern: ["увлажнение", "чувствительная кожа"],
  shortDescription: "Короткое описание для карточки.",
  accent: "#e9dfd7"
}
```
